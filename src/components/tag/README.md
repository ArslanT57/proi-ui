[proi-ui](https://github.com/specialdoom/proi-ui)

###### Tag component

### Usage

```sveltehtml
<script>
import { Tag } from 'proi-ui'
</script>

<Tag outlined>Tag</Tag>
<Tag color="#3a5asd">Tag</Tag>
```

### API

| Property | Description                 | Type    | Default |
| -------- | --------------------------- | ------- | ------- |
| outlined | Whether the tag is outlined | boolean | false   |
