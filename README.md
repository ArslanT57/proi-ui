proi-ui
---------
> proi UI Library for Svelte framework.  
---

## Installation
All controls
```bash
npm i proi-ui
```

## Usage
```javascript
<script>
import { Button, Input } from 'proi-ui'

function click () {
    console.log('clicked');
}
</script>

<div>
    <Input label="My label" squared />
    <Button on:click={click}/>
</div>
```
